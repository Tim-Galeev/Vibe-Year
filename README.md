# Новогодний Заброс (Santa's Chimney Dash)

Новогодний 2D автораннер для Godot 4.5

## Описание

Дед Мороз на санях развозит подарки, бросая их в дымоходы домов. 
Летите как можно дальше, набирая очки и избегая препятствий!

## Управление

- **Мышь (тянуть вниз)** - прицеливание
- **Мышь (отпустить)** - бросить подарок
- **Пробел / W / Стрелка вверх** - прыжок
- **E** - использовать шампанское (ускорение)
- **Q** - бросить ёлку-гранатомёт

## Механики

### Система очков
- Попадание в трубу: +100 очков + восполнение подарка
- Уничтожение мандарина-затычки: +50 очков

### Препятствия типа А (для саней)
- **Гора мандаринов** - подбрасывает сани, сбивает прицел
- **Озеро оливье** - замедляет сани на 1.5 секунды

### Препятствия типа Б (для труб)
- **Мандарин-затычка** - блокирует трубу, уничтожается подарком

### Бустеры
- **Шампанское** - собирается в полёте, даёт ускорение (макс 3)
- **Ёлка-гранатомёт** - заряжается пассивно, уничтожает препятствия

```
│   ├── chimney.tscn           # Дымоход

│   ├── mandarin_pile.tscn     # Гора мандаринов
│   ├── olivie.tscn            # Озеро оливье
│   ├── champagne_pickup.tscn  # Бутылка шампанского

│   ├── game_manager.gd        # Глобальный менеджер игры
│   ├── santa_sleigh.gd        # Сани Деда Мороза
│   ├── roof_spawner.gd        # Генератор крыш
│   ├── roof.gd                # Логика крыши
│   ├── chimney.gd             # Логика дымохода
│   ├── gift.gd                # Логика подарка
│   ├── mandarin_blocker.gd    # Мандарин-затычка
│   ├── mandarin_pile.gd       # Гора мандаринов
│   ├── olivie.gd              # Озеро оливье
│   ├── champagne_pickup.gd    # Подбор шампанского
│   ├── tree_grenade.gd        # Ёлка-гранатомёт
│   ├── background.gd          # Фоновый скроллер
│   └── hud.gd                 # Интерфейс
└── project.godot    # Настройки проекта
```

## Запуск

1. Откройте проект в Godot 4.5+
2. Godot автоматически сгенерирует UID для всех файлов
3. Нажмите F5 для запуска
